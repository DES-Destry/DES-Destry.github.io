File protect

Creator: DES Destry(Unimaster presents)

Description: The application is written in C# using WPF technology. The code is written using the MVVM architectural pattern. The application is designed to encrypt and subsequently decrypt folders. To take action, you must enter your password (created when you first start the application). The password is encrypted in the settings file. There are many settings for regulating the actions of this application. It is possible to send a report by mail. The report contains error files (my personal file permission), logs, settings files, and a history of user actions. The report may also contain your comment and your mail to receive a response. Starting with version 0.9.5, there is now an update check. After checking, if the version of your application is not the latest, then the application updates itself and deletes the old itself on subsequent launch.
